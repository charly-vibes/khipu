khipu is a Python tool-agnostic agent trace analyzer. Reads conversations/traces/logs, outputs workflows, patterns, and crystallization candidates. Three interfaces from one codebase: CLI (typer), MCP server, Python library. Five pipeline stages: ingest -> redact -> condense -> analyze -> emit. Extensible via drop-in Python files (ingestors), prompt markdown files (analyzers), Jinja2 templates (emitters), TOML configs (backends). Built-in ingestors: claude_code (JSONL), aider (.aider.chat.history.md), generic (markdown fallback). Built-in analyzers: workflows, patterns, crystallize (DAG with depends_on). Built-in emitters: markdown, json, claude_md. LLM backends: API mode (HTTP) or CLI mode (shell out to ollama, claude CLI, llm CLI). Redaction is on by default. Condensation is deterministic by default (auto mode). Tech stack: Python>=3.11, typer, jinja2, httpx, mcp. Dev: pytest, ruff, mypy. Install: uv tool install khipu.
